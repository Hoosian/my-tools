# my-tools

Personal collection of handy tools.

## Tools

### 1. js-excel-generator
Excel Test Data Generator (Python)

Quickly generate Excel files with random data, custom headers, and multiple data types.

```bash
cd js-excel-generator
pip install -r requirements.txt
python -m src.main generate test.xlsx --count 100
```

See [js-excel-generator/README.md](js-excel-generator/README.md) for details.

---

### 2. python-image-cropper
Node.js Image Format Converter

Batch convert images between formats with configurable quality.

```bash
cd python-image-cropper
npm install
npm start
```

See [python-image-cropper/README.md](python-image-cropper/README.md) for details.
